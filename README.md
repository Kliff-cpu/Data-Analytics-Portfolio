# Data Analytics & Quantitative Finance Portfolio

Welcome to my portfolio. This repository provides a curated overview of projects demonstrating my skills in data analysis, statistical modelling, financial risk, and quantitative finance.

The work spans time series modelling, portfolio risk measurement, and derivatives valuation, combining both theoretical understanding and practical implementation.

---

## 🔹 Projects

### 1. Time Series Modelling and Forecasting  
🔗 https://github.com/Kliff-cpu/Time-Series-Modelling-and-Forecasting  

This project applies ARIMA and GARCH-type models to analyse and forecast time-dependent data. It includes model comparison using statistical metrics and the Diebold–Mariano test for forecast evaluation.

**Focus:**
- Volatility modelling  
- Forecasting accuracy  
- Time series econometrics  

---

### 2. Portfolio Risk Measurement and Optimization  
🔗 https://github.com/Kliff-cpu/Portfolio-Risk-Measurement-and-Optimization  

This project evaluates portfolio risk using multiple downside and tail-risk measures, including VaR, CVaR, EVaR, and CDaR. It also incorporates credit risk into the overall analysis.

**Focus:**
- Tail risk analysis  
- Portfolio optimization  
- Risk measure comparison  

---

### 3. Interest Rate Swap Valuation (Excel Model)  
🔗 https://github.com/Kliff-cpu/Interest-Rate-Swap-Valuation-in-Excel  

This project builds an Excel-based model for valuing a bullet interest rate swap, solving for the fair fixed rate and analysing sensitivity using PV01.

**Focus:**
- Fixed income derivatives  
- Present value modelling  
- Sensitivity analysis (PV01)
